# Stone Paper Scissor Game (Python)

This is a simple command-line Stone Paper Scissor game written in Python where a user plays against the computer.

## Features

- 10 rounds gameplay
- Random computer choices
- Score tracking
- Displays win, lose, and draw results

## Technologies Used

- Python
- Random module

## How to Run

1. Install Python
2. Run the program:

python stone_paper_scissor.py

3. Enter your choice:
1 = Stone  
2 = Paper  
3 = Scissor

## Purpose

This project was created to practice Python programming concepts like loops, conditions, dictionaries, and exception handling.
